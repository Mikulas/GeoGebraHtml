Documentation

  • Position - generic x;y object
  • * Slope - different implementation of Position
  • Container - holds Elements
  • Element - generic geometric object
  • * Point - describes one point on 2D plane; two points with same position might exist
  • * Line - determined by either two Points or a Point and Slope
  • * Circle - determined by either two Points or a Point and radius
